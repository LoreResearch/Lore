# Contributing

## Local Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Contribution Rules

- keep scoring changes separate from memo-copy edits
- update tests when governance, runway, or unlock logic changes
- keep README and runbook language focused on token underwriting

## Pull Request Notes

- explain which diligence dimension changed
- include a sample memo or ranking delta
- update docs when the buyer workflow changes
