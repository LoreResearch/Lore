# Lore Runbook

## What Lore Is For

Lore helps a token buyer answer whether a protocol deserves capital after accounting for governance, runway, unlock pressure, and fee retention.

## Daily Operator Loop

1. Run `npm run dev`.
2. Read the research board for the current memo.
3. Compare the protocol strip against the nearby alternatives.
4. Escalate only the protocols where demand quality can absorb expected supply.

## Questions Lore Must Answer

- who controls the token and governance surface
- how long the treasury can keep shipping
- how much supply is coming into the market next
- whether protocol usage is monetizing into something durable
